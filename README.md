# 🚀 AI Travel - 微信小程序

一款专为旅行爱好者设计的智能旅行规划微信小程序，提供AI推荐路线、景点介绍、行程管理等功能。

## ✨ 特性

- 🎯 **智能推荐** - AI驱动的个性化路线推荐
- 🗺️ **行程规划** - 简单易用的行程创建和管理
- 📍 **热门景点** - 精选旅游景点和详细介绍
- 👤 **个人中心** - 个人信息管理和历史记录
- 📱 **原生体验** - 微信小程序原生交互体验
- 🎨 **现代化UI** - 美观的界面设计和流畅动画

## 🏗️ 技术架构

### 小程序技术栈
- **微信小程序原生开发** - 使用WXML、WXSS、JavaScript
- **组件化开发** - 模块化的代码结构
- **响应式设计** - 适配各种手机屏幕尺寸

### 项目结构
```
miniprogram/
├── app.js                 # 小程序入口文件
├── app.json              # 小程序配置文件
├── app.wxss              # 小程序全局样式
├── project.config.json    # 项目配置文件
├── sitemap.json          # 站点地图配置
├── pages/                # 页面目录
│   └── index/           # 首页（单页面应用）
│       ├── index.js      # 页面逻辑
│       ├── index.wxml    # 页面结构
│       ├── index.wxss    # 页面样式
│       └── index.json   # 页面配置
└── README.md           # 项目说明文档
```

## 🚀 快速开始

### 环境要求
- 微信开发者工具
- 微信小程序开发账号

### 安装和运行
1. 克隆项目到本地
2. 打开微信开发者工具
3. 导入项目，选择 `miniprogram` 目录作为项目根目录
4. 填入你的小程序AppID
5. 点击编译运行

## 📱 功能页面

### 1. 首页
- **欢迎界面** - 个性化的欢迎信息
- **探索地图** - 交互式地图展示
- **AI推荐** - 智能路线推荐轮播
- **精选目的地** - 热门旅行目的地展示

- **行程列表** - 查看和管理所有行程

- **景点列表** - 精选旅游景点展示
- **景点评分** - 用户评分和评论系统

### 4. 个人主页
- **用户信息** - 个人资料和头像
- **统计信息** - 旅行次数和收藏统计
- **我的行程** - 个人历史行程记录
- **设置选项** - 应用设置和帮助信息
